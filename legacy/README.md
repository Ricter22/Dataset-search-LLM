# Legacy Material

This directory contains archival notebooks and exploratory files kept for reference only.

- `legacy/notebooks/`: root-level experiment notebooks and their helper utility module.
- `legacy/target/`: notebook-based TARGET experimentation and preprocessing runs.
- `legacy/benchmark/`: archival benchmark notebook material retained outside the main public surface.
- `legacy/ntcir15/`: notebook-only NTCIR-15 exploratory material.

Nothing under `legacy/` is the recommended public entrypoint for reproducing the repository structure.
